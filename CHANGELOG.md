# Changelog

## 0.1.0-alpha.1

Initial public Alpha release preparation.

### Supported

- Python scripts and simple projects
- Node.js scripts and simple projects
- TypeScript single-file scripts, with limited directory support
- Global Capsule cache under `~/.capsule`
- `run`, `scan`, `inspect`, `lock`, `doctor`, and `clean` commands

### Notes

- Capsule Alpha is intentionally scoped to Python, Node.js, and TypeScript.
- Go, Rust, and Ruby are roadmap items, not active functionality.
- Tested primarily on Windows during Alpha development.
